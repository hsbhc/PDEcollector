$$Kuramoto-Sivashinsky (KS) equation$$
$$ u_t = - u u_x - u_{xx} - u_{xxxx}$$
$$x\in [-10,10] ,t \in [0,50]$$
$$u(0,x) = -\sin(\pi x/10)$$
$$u(t,-10) = u(t,10)$$
$$u_x(t,-10) = u_x(t,10)$$
$$u_{xx}(t,-10) = u_{xx}(t,10)$$
$$u_{xxx}(t,-10) = u_{xxx}(t,10)$$

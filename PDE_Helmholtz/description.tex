$$\Delta u(x,y)+k^{2}u(x,y)=q(x,y)$$

$$u(x,y) = u(x,y)\ \ (x,y) \in \partial \Omega$$
$$q(x,y)=-(a_{1}\pi)^{2}u-(a_{2}\pi)^{2}u+k^{2}u$$\\
$$u(x,y) = sin(a_{1}\pi x)sin(a_{2}\pi y)$$
